# MayaScripts

Maya Python 스크립트 툴 모음. 리깅 중심으로 구성.

## 폴더 구조

| 폴더 | 설명 |
|------|------|
| `rigging/` | 리깅 관련 툴 (Skin / Set / Joint / Naming / Utility) |
| `pipeline/` | 임포트/익스포트, 파일/에셋 관리 툴 (추후 추가) |
| `utils/` | 공통 함수, 헬퍼 유틸리티 |

## 실행 방법

`rigging/shelf_YG_rigging.py` 내용을 Maya Script Editor에서 실행하면 셸프 버튼이 등록된다.
